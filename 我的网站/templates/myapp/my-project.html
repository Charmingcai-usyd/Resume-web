3D printer based on IoT

Project duration : about 3 months

Background : Graduation design project

Achievement :
  • Excellent graduation project in Uni
  • Utility model patent certificate
  • Participate in the selection of Excellent graduation project in Hubei Province

    • This project attempts to design a 3D printer data monitoring and surveillance system using Linux as the operating system, attempting to solve many of the process control problems that occur with closed-loop control systems for 3D printers: step losses in stepper motors, plugged nozzles ect. It also overcomes the problems of traditional analogue CCTV monitoring systems with short transmission distances, inability to network and large storage media. By designing a networked monitoring system, users can access the data monitoring platform from any Internet port after accessing Ethernet at the site. The platform automatically completes image identification and data processing and analysis, and provides alarm functions at the same time.

    PMSM current predictive control based on disturbance torque observer

    Project duration : about half a year

    Background : Research topics

    Achievement :
  • Published in 2021 China Automation Congress (CAC) which is the top 3 conferences in China's control fields
  • IEEE published in 14 March 2022

    • Permanent magnet synchronous motor (PMSM) is easy to be affected by torque ripple and dead time of inverter when it runs at medium and low speed, resulting in poor control characteristics of PMSM. In order to solve this problem, this paper takes PMSM in medium and low speed as the research object, and replaces the traditional PI control with current predictive control. At the same time, a disturbance torque observer is introduced to estimate the voltage error caused by the disturbance torque of the motor model in real time, and feedback to the input terminal to counteract the impact of the disturbance on the motor. Finally, through MATLAB / Simulink and experiments, it is verified that the proposed scheme can effectively improve the dynamic characteristics of the motor, suppress external interference and improve the control performance of the motor.

    Construction of a PMSM model

    Simulation with matlab/simulink software

    Constructing PMSM mathematical model.

    Analysing current predictive control.

    Adding PMSM disturbance torque disturbance observer.

    Simulation Analysis of new module.

    Laboratory tests on PMSM control strategies

    Build real circuits to test performance

    Actual waves are the same as the simulation waves.

    Better performance than traditional PI control strategies.

    The paper's conclusions are correct and can be reproduced in the real world.
